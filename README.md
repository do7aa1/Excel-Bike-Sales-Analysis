# Bike Buyers Data Analysis (Excel Project)

## Project Overview
This project analyzes customer data to understand factors affecting bike purchases.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Dashboard Visualization

## Key Insights
- Income influence on purchases
- Age group buying patterns
- Regional trends

## Files
- Dataset: Raw customer data
- Dashboard: Interactive Excel dashboard

## Outcome
Created a business-style dashboard to support decision making.
